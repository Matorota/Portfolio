A modern portfolio website showcasing my development projects and skills. Built with React, TypeScript, and Vite for optimal performance.

## 📋 About

This portfolio displays 6 major projects across different technologies including:

- **Full-Stack Web Development** (React, Node.js, Express)
- **Desktop Applications** (C#, WinForms)
- **Security & Encryption** (C#, .NET)

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Matorota/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🌐 Development

```bash
# Start dev server (runs on http://localhost:5173)
npm run dev

# Build production
npm run build

# Preview production build
npm run preview

# Run ESLint
npm run lint
```
